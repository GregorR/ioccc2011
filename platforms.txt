Shown is the exact compiler version tested. All tests are on GNU/Linux unless
specified otherwise. I expect things to generally work with other versions of
the compilers, but show only the tested version for honesty sake.

gcc 4.6.2 + x86[_64]:
    -O0, -O1, -O2, -O3

owcc 1.9 + x86 + Windows:
    -fno-stack-check must be provided in all modes

gcc 4.4.5 + alpha, arm, powerpc, powerpc64, s390x (IBM z/Series):
    -O0, -O1, -O2, -O3

gcc 4.4.5 + mips, mipsel:
    -O0
    -O1
    -O2 -fno-optimize-sibling-calls
    -O3 -fno-optimize-sibling-calls

gcc 4.4.5 + sh4:
    -O0
    Higher optimization levels don't work for unknown reasons.
